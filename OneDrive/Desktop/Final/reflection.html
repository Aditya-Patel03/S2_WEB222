Final Assessment - Reflection

Explain the concept of Responsive Design. What does the term mean?

  • Responsive design is a GUI design (Graphical User Interface) approach used to create content that can adjust according to their screen size smoothly.
  • Designers can resize elements in relative units and apply media requests/queries

How does responsive design relate to web technologies?

  • The Responsive design has three vital components/principles:
  • 1.Fluid Grid System: Elements occupy the same percentage of space regardless of the screen size.
  • 2.Fluid Image Use: Images aren't fluid by nature. This results the design having inconsistency across devices as default images will fail to adjust and hence display is out of proportion. To solve this we can use img {max-width: 100%;} in css file.
  • 3.Media Queries: Filters used to detect browsing device's dimensions to make the design appear nicely. These can alter the size layout to meet certain requirements.

What did you learn about building web apps for different devices through your research?

  • Web apps are built using Javascript, CSS, HTML, and Python. Unlike Mobile applications we are not required to have any standard development kit. Although developers need to access templates. Web apps are considered to be built quicker and easier.
